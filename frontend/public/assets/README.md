# School Branding Assets

## Logo Files

Place the Success City Academy logo files in this directory:

- `logo.png` - Full color logo for light backgrounds (required)
- `logo-white.png` - White version for dark backgrounds (optional)

## Logo Specifications

- Format: PNG with transparent background
- Recommended size: 200x200px minimum
- Colors: Green (#16a34a) and Yellow (#eab308)

## Usage

The logo will be used in:
- Navigation header (all pages)
- Login page
- PDF report headers
